# Package index

## Create Stickers

The core functions for building hex stickers.
[`hex_sticker()`](https://cttir.github.io/hexmakR/reference/hex_sticker.md)
returns a ggplot object;
[`hexmakr_app()`](https://cttir.github.io/hexmakR/reference/hexmakr_app.md)
opens the interactive Shiny designer.

- [`hex_sticker()`](https://cttir.github.io/hexmakR/reference/hex_sticker.md)
  : Create a Hex Sticker
- [`hexmakr_app()`](https://cttir.github.io/hexmakR/reference/hexmakr_app.md)
  : Launch Interactive Hex Sticker Designer

## Themes & Icons

Explore the 19 built-in color themes and 46 built-in icons. Each theme
provides coordinated `dark` and `light` color palettes.

- [`hexmakr_themes()`](https://cttir.github.io/hexmakR/reference/hexmakr_themes.md)
  : List Available Themes
- [`hexmakr_icons()`](https://cttir.github.io/hexmakR/reference/hexmakr_icons.md)
  : List Available Icons
- [`hexmakr_preview_theme()`](https://cttir.github.io/hexmakR/reference/hexmakr_preview_theme.md)
  : Preview a Color Theme

## Export

Save hex stickers to PNG (transparent background) or SVG.

- [`save_hex()`](https://cttir.github.io/hexmakR/reference/save_hex.md)
  : Save a Hex Sticker
- [`save_hex_svg()`](https://cttir.github.io/hexmakR/reference/save_hex_svg.md)
  : Save a Hex Sticker as SVG
